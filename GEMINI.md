# Winamp Clone - Project Brain

## Project Summary
A pure-file-based MP3 player inspired by Winamp. Built with Vanilla JS and high-end CSS aesthetics.

## Tech Stack
- **Frontend**: HTML5, Vanilla JavaScript, CSS3
- **Audio API**: Web Audio API
- **Visuals**: Canvas API for visualizers
