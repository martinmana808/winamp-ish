# Winamp Clone - Project Brain

## Project Summary
A native macOS MP3 player inspired by the classic Winamp. Built with Swift, SwiftUI, and AVFoundation.

## Tech Stack
- **Language**: Swift 6
- **Framework**: SwiftUI
- **Audio API**: AVFoundation
- **UI Architecture**: MVVM
